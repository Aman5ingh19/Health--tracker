# Health Tracker

docodev.AI - An AI-powered health tracking and symptom analysis web app.  
Built with the MERN stack + integrated symptom checker API.

## Features
- Check BP, sugar, BMI, and heart rate
- AI-powered symptom diagnosis
- Interactive human body animation
- Multi-language support
- Doctor recommendations

## Tech Stack
- React, Node.js, Express, MongoDB Atlas
- RapidAPI (Symptom Checker)
- Git, GitHub
